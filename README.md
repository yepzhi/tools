# OSINT Social Analyzer

A web-based tool to search for profiles across 1000+ social media platforms.

## Features
- Search by Username, Email, or Phone
- Real-time analysis
- Dark mode UI with glassmorphism design

## Backend Status
The backend is currently hosted on **Hugging Face Spaces**: `https://yepzhi-tools.hf.space`

## Setup
1. Clone the repository
2. Open `index.html` in your browser (or visit the GitHub Pages link)

## Usage
Enter a username and click Search. The app will connect to the live backend to fetch results.
